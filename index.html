生命，短一些也可以

发表于 2016-06-30   |   分类于 感悟

在我们这么多的祝贺语当中，有一个祝贺语使用频率非常高，而且常用在祝贺老年人身上，这句祝贺语叫“祝您长命百岁”。
是啊，我们当中有些人真的非常希望能够活到一百岁，甚至千岁万岁。所以古代很多皇帝都在寻找永生之药。
这种希望自己活得很久的观念，应该说是动物的求生本能在驱使着我们不能死，要生存下去。然而，很多时候，我们虽然活着，但活得很痛苦，这恐怕就是人们所说的生不如死吧。
可能有些人会问自己，我活着是为了什么，我没有活得像大家期望的那样，怎么办？
其实每个人都应该找到真正属于自己的独特的人生意义，哪怕这是一件很小的事情，有一天自己真的实现了，那么人生的意义也多了一分，当这些小意义累计起来，我们的人生将是满分的人生。

随想

发表于 2016-05-20   |   分类于 思考

我可以参透所有道理，却无法参透“爱”和“情”。
人生的路很短暂，也很漫长。有时候，总觉得时间怎么过得那么快，来不及好好口味。有时候，却希望时间快点过去，好结束煎熬。
以前，总希望自己能有一种技能，可以回到过去，重新走一遍曾经走过的路，把不好的过去纠正过来，重新走上美好的人生。有时候，总会抱怨自己，当初为什么这么傻，当初为什么要那样，我们自责，我们内疚，我们恨不得一死了断。人生的路存在各种可能性，我们永远无法知道每一条路通往何方，也无法知道每一条路的结局是否如你所愿。我们走人生的路，并不是像走迷宫，走错了一条路，返回再走另一条路。当我们选择了其中一条路的时候，就应该相信自己的选择，不后悔自己的选择，坚持自己的选择，遇到困难想办法解决。人生没有多少次重来的机会，有时，错过了，就是一辈子了。

K12教育APP产品分析

发表于 2016-05-01   |   分类于 产品经理

目前常用的APP

  • 学霸君
  • 猿题库
  • 作业帮
  • 学习宝
  • 微课圈（2015年9月上线）

###目前K12产品切入点
课程、答疑、测试

微课圈

  • 四大版块： 精品课程、海量题库、答疑解惑、专题提升
    学生通过“精品课程”模块搜索并完成自己需要的课程；听完课后可以直接跳转到“海量题库”模块，通过习题、真题检验学习成果；遇到疑难困惑进入“答疑解惑”模块后能与在线名师实现线上交流，得到解答和思路点拨，从而完成一个完整的学习过程。
  • 特点：集中当下领域内通用产品的所有功能。 致力于开发学生潜能的专题课
  • 目前发展状况：微课圈自今年6月初内测版本走进北京四中、八中、101中学、人大附中、师大附中等10所名校，7月覆盖天津、山西等多所中学，在尚未正式上线的阶段已赢得广大师生的好评。7月25日，微课圈在哈尔滨举行项目说明会，迅速打开东北市场。

18岁以下学生的特点

  • 缺乏耐性

K12在线教育产品的刚需

  • 帮助学生更好好地学习、更有效地吸收理解知识、又快又准的做题、更准确地报志愿等
  • 帮学生提高效率、减少负担、更轻松愉快地学习

需要注意的

  • 产品必须符合人性： 贪婪、懒惰、嫉妒、虚荣、好色、从众、自卑。互联网题库产品因为他们和用户的“懒惰”作对，所以半死不活。

所谓的梦想

发表于 2016-05-01   |   分类于 思考

以前家里穷，当看见邻居家装上固定电话后，心里面就有个梦想，希望有一天，家里也能装上固定电话，需要用电话的时候只能跑到邻居家。

上学后，当看到同学家的父母开着小车接送他们上学的时候，心里面就有个梦想，希望有一天，自己家也能开上小车，想去哪就去哪。

我有很多梦想，很多很多！

自从学习佛法之后，我发觉，以前所谓的梦想，不过是自己的贪念心罢了。

我的“梦想”只不过是希望拥有某一样东西罢了！

所谓的选择

发表于 2016-05-01   |   分类于 感悟

昨天在深圳书城遇到一个中年男人，其实我已经留意他有一段时间了。之前发现他会主动地去找年轻人去聊天，有男的，也有女的。聊的话题之一就是如果赚更多的钱。就在昨天，他终于逮到我了。当他坐在我旁边的时候，我已经开始准备如何接招。刚开始的聊天还算正常，无外乎就是问我做什么工作的呀，他自己 一些个人情况什么 。后来发现我的回应比较冷淡，他也只好生硬的提议说：给他一分钟的时候，让他对我的人生进行改变。我看了一下他随身推着的手推车，车上有一本投资类的书，我一下就知道他应该就是想找我投资的人（P2P啊，大宗商品啊等等，后来果然如我所料）。
我跟我说的第一个话题是“选择”。他问我：你手掌最长的是哪个手指。我说：当然是中指。然后他举了例子说：姚明的手指比你的手指长多了，其实人生又很多更好的选择，不能只看到自己手中的选择。（此处省略几百字……）
对于这个话题，我是相当有把握能辩赢他的！我说：姚明的手指并不属于我，我无法选择他的手指。正如我并不能选择成为王健林的儿子、王思聪的弟弟。我唯一可以选择的是我自己的五个手指。我们总是想着有更好的选择，而忽视了我们已经拥有的东西，吃着碗里看着锅里。没错，可能你拥有的并不是最好的，但可能是最适你的，你何不把他的作用发挥到极致的。还有，如果你拥有的这“五个手指”已经足够让你这辈子过得幸福快乐，你何必去追求第六个“手指”呢。有些人是贪心，不知足，总想得到更多，害怕失去，所以这辈子都过得不幸福快乐。而珍惜自己拥有 ，坦然地面对得与失，将拥有更广阔的人生。
